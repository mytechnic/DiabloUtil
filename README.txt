프로그램명 : 우버디아 스타헌터 (Diablo Clone Star Hunter)

이 프로그램은 ...
- 프로그래머 직업 윤리를 지키며 개발하였습니다.
- 악성 코드를 포함하지 않습니다.
- 데이터를 외부로 전송하지 않습니다.

1. 기능 소개
    * 게임IP 조회
        목표 IP(입력)
            '목표 아이피'는 컴마(,)로 구분하여 1개 이상 입력 가능하며, 검색 대상의 IP를 입력 할 수 있습니다.
        모드 설정
            '수동검색 모드'는 '아이피 검색' 버튼을 통해 아이피를 조회 할 수 있습니다.
            '자동검색 모드'는 실시간으로 게임 아이피를 조회합니다.
            '지킴이 모드'는 현재 입장한 게임을 벗어 날 경우 다른 모드를 변경하기 전까지 계속 알람이 발생합니다.
        목표 IP(텍스트)
            검색 결과가 나온 '목표 IP'
        지역
            현재 접속한 지역을 의미합니다.
        게임방 IP
            현재 접속된 방을 의미합니다.
        지킴이 IP
            지키고자 하는 방의 IP를 의미합니다.
        TCP/IP 정보
            디아블로 프로그램과 연결된 모든 IP 정보를 표시합니다.
        타이머
            최초 게임방 IP를 발견 한 순간부터 1초씩 증가합니다.
        전광판
            게임방 IP 정보와 타이머 정보를 유사 오버레이로 표현합니다.
        초기화
            현재 검색된 IP의 결과 정보를 초기화 합니다.

    * 방화벽 설정
        목표 IP(입력)
            1개의 아이피만 입력 가능합니다. 추후 머리가 맑아지면 멀티가 지원되도록 적용하겠습니다.
        차단 정책
            아이피 좁히는 범위를 결정 할 수 있습니다.
        적용 결과
            '방화벽 적용' 버튼을 누른 결과가 나타납니다.
        방화벽 적용
            목표 IP에 맞는 윈도우 방화벽을 설정합니다. (+ 기존 설정된 방화벽 삭제 기능)
        방화벽 삭제
            적용한 방화벽 정보를 삭제합니다.
        방화벽 조회
            윈도우의 방화벽 설정 화면을 오픈 할 수 있습니다.

    * 전광판 설정
        전광판 위치
            X, Y 좌표를 통해 전광판의 위치를 실시간으로 변경 할 수 있습니다.
        타이머 설정
            전광판의 타이머를 보이게 하거나 감출 수 있습니다.
        일반/오류 폰트/색상 설정
            목표 IP를 설정하지 않았거나, 오류 메시지의 스타일을 지정 할 수 있습니다. 오류의 경우에는 메시지을 설정 할 수 없습니다.
            일반/성공/실패 메시지는 {{GAME_IP}} 코드를 통해 게임방 IP를 표현 할 수 있습니다.
        성공시 폰트/색상 설정
            목표 IP가 일치 했을 경우의 메시지를 설정 할 수 있습니다.
        실패시 폰트/색상 설정
            목표 IP가 일치하지 않을 경우 메시지를 설정 할 수 있습니다.

2. 특징
    디아블로 우버디아 이벤트에서 편리하게 사용 할 수 있습니다.
    디아블로 프로세스에 접근하지 않아 안전하게 사용 할 수 있습니다.

3. 다운로드
    제작자는 이 프로그램 소지/사용을 통해 발생하는 그 어떤 사유에 대해서도 책임을 지지 않습니다.
    동의하지 않을 경우에는 이 프로그램 다운로드 및 사용을 금지합니다.
    디스코드 초대코드 - https://discord.gg/6s328VEE

4. 실행방법
    압축을 풀고 폴더 안의 DiabloCloneStarHunter.exe 파일을 '관리자 권한'으로 실행 해 주세요.
    능력 부족으로 바로가기 아이콘을 만들진 못했습니다.

5. 사운드 변경 방법
    수동검색/자동검색 모드에서 목표 IP가 일치 할 경우 - 설치 폴더에 'ok.mp3' 파일을 담아두면, 성공시 경우 해당 음악으로 1회 재생 됩니다.
    지킴이 모드에서 'stay_out.mp3' 파일을 넣어 둘 경우 해당 음악으로 재생 됩니다.

[사용방법 요약]
    1. 디아블로 클라이언트를 실행합니다.
    2. 관리자권한으로 이 프로그램을 실행합니다. 1번과 순서가 바껴도 상관 없습니다.
    3. 목표 아이피(IP)를 입력합니다.
    4. 방화벽 설정 메뉴에서 방화벽을 설정합니다.
    5. 게임IP 조회 메뉴에서 자동검색 모드로 선택합니다. (실시간 적용)
    6. 전광판 클릭(토글 방식 - ON/OFF) 합니다.
    7. 디아블로 게임을 생성한다. 미리 생성해도 상관 없습니다.
        생성 성공 - 최초로 게임IP 대역이 원하는 대역과 성공하면 방화벽 삭제하고 다음 단계로 진행합니다.
        생성 실패 - 배틀넷 및 디아블로 종료 후 실행하여 7번 단계를 다시 수행합니다.)
    8. 목표 아이피(IP)가 일치 할 때까지 게임을 계속 생성합니다.
