프로그램명 : 우버디아 스타헌터 (Diablo Clone Star Hunter)

이 프로그램은 ...
- 프로그래머 직업 윤리를 지키며 개발하였습니다.
- 악성 코드를 포함하지 않습니다.
- 데이터를 외부로 전송하지 않습니다.

1. 게임IP 조회
    '목표 아이피'는 컴마(,)로 구분하여 1개 이상 입력 가능합니다.
    '수동검색 모드'는 '아이피 검색' 버튼을 통해 아이피를 조회 할 수 있습니다.
    '자동검색 모드'는 실시간으로 게임 아이피를 조회합니다.
    '지킴이 모드'는 현재 입장한 게임을 벗어 날 경우 다른 모드를 변경하기 전까지 계속 알람이 발생합니다.
    'IP 로그 초기화'는 하단의 로그를 지웁니다.
    '전광판'은 메인 화면의 오버레이와 유사한 기능을 제공하며, 아이피 및 타이머 정보를 확인 할 수 있습니다.

2. 방화벽 설정
   방화벽 설정 기능은 '목표 IP' 입력이 필수이며, 1개의 아이피만 입력 가능합니다.
    '차단 정책'을 통해 아이피 좁히는 범위를 결정 할 수 있습니다.
    '방화벽 적용' 기능을 통해 설정에 맞는 방화벽을 새로 설정합니다.
    '방화벽 삭제' 기능을 통해 이 프로그램을 통해 설정한 방화벽을 삭제 할 수 있습니다.
    '방화벽 조회' 기능을 통해 윈도우의 방화벽 설정 화면을 오픈 할 수 있습니다.

3. 특징
    디아블로 우버디아 이벤트에서 편리하게 사용 할 수 있습니다.
    디아블로 프로세스에 접근하지 않아 안전하게 사용 할 수 있습니다.

4. 다운로드
    제작자는 이 프로그램 소지/사용을 통해 발생하는 그 어떤 사유에 대해서도 책임을 지지 않습니다.
    동의하지 않을 경우에는 이 프로그램 다운로드 및 사용을 금지합니다.
    디스코드 초대코드 - https://discord.gg/6s328VEE

5. 실행방법
    압축을 풀고 폴더 안의 DiabloCloneStarHunter.exe 파일을 '관리자 권한'으로 실행 해 주세요.
    능력 부족으로 바로가기 아이콘을 만들진 못했습니다.

6. 사운드 변경 방법
    수동검색/자동검색 모드에서 목표 IP가 일치 할 경우 - 설치 폴더에 'ok.mp3' 파일을 담아두면, 성공시 경우 해당 음악으로 1회 재생 됩니다.
    지킴이 모드에서 'stay_out.mp3' 파일을 넣어 둘 경우 해당 음악으로 재생 됩니다.

7. 전광판 위치 변경 방법
    DiabloCloneStarHunterConfig.yaml 파일을 열어, dashboardPosX, dashboardPosY 값을 수정하면 원하는 위치에 표시 할 수 있습니다.

[사용방법 요약]
    1. 디아를 킨다.
    2. 관리자권한으로 이 프로그램을 실행한다
    3. 목표 아이피를 입력한다.
    4. 방화벽 설정 메뉴에서 화면에서 방화벽을 적용한다.
    5. 게임IP 조회 메뉴에서 자동검색 모드로 선택한다.
    6. 전광판 클릭(토글 방식 - ON/OFF)
    7. 디아블로 게임을 생성한다.
        생성 성공 - 설정 한 방화벽 삭제
        생성 실패 - 디아블로 재실행 후 다시 게임을 생성)
    8. 목표 아이피가 맞을 때까지 게임 계속 생성
